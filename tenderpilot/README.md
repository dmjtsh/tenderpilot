# TenderPilot — парсер тендеров ЕИС

Скрипт автоматически скачивает XML-архивы тендеров с сайта ЕИС (zakupki.gov.ru), парсит их, фильтрует по параметрам компании и сохраняет в SQLite и CSV.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка под свою компанию

Откройте `config.py` и измените:

- `TARGET_OKVED` — коды ОКВЭД вашей деятельности
- `TARGET_REGIONS` — коды регионов (например `"77"` — Москва, `"78"` — Санкт-Петербург)
- `MIN_PRICE` / `MAX_PRICE` — диапазон суммы контракта
- `REGION_FTP_NAMES` — добавьте ваши регионы (названия директорий на FTP ЕИС)

## Запуск

```bash
cd tenderpilot
python eis_parser.py
```

## Файлы результатов

- `tenderpilot.db` — SQLite база со всеми найденными тендерами
- `tenders_output.csv` — CSV для Excel/таблиц
- `sample_tender.xml` — пример одного XML-тендера (для следующего шага с AI-анализом)
- `parser.log` — лог ошибок и процесса

## Структура БД

Таблица `tenders` содержит все поля тендера + `is_analyzed` и `ai_verdict` для последующего AI-анализа.
